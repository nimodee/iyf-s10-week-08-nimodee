## Project Description
A React (Vite) mini-project that simulates a simple community posting app. Users can create, view, like, delete, and search posts. The project was built to practice core React concepts like components, props, state management, and event handling.

## Technologies Used
- React
- Vite
- JavaScript
- CSS

## Features
- Create posts
- Display list of posts
- Like posts
- Delete posts
- Search/filter posts
- Reusable components (Button, Card, Layout)
- Basic stats (total posts and likes)

## How to Run
1. Clone this repository
2. Open project folder in terminal
3. Run npm install
4. Run npm run dev

## Lessons Learned
I learned how to structure a React project using components, manage state using useState, pass data using props, and handle user interactions like creating, liking, deleting, and filtering posts.

## Challenges Faced
I faced issues with props passing between components, especially when handling delete and like functions across multiple components. I solved this by lifting state up to the App component and properly passing functions through PostList to PostCard.
